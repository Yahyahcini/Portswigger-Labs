# Authentication
## Overview
Authentication vulnerabilities occur when an application fails to properly verify the identity of a user, allowing attackers to gain access to accounts and functionality they shouldn't have.
## Types of Authentication Vulnerabilities
- Username enumeration
- Password brute force
- Multi-factor authentication bypass
- Password reset flaws
- Session-based vulnerabilities
